## Quick Start

### Run Locally
```bash
cd frontend
npm install
npm run dev
# Visit http://localhost:5173
```

### Deploy to GitHub Pages
1. Enable GitHub Pages in repository settings
2. Select "GitHub Actions" as source
3. Push to main branch
4. Workflow will auto-deploy
